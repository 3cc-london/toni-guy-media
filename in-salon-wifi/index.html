in-salon wifi

In-salon Wi-Fi has been provided by O2 for the last year and is a free, premium wifi service that is open to everyone regardless of the mobile network they are on. The peak figure in December 2014 was 375,000 clients using the network for an average of 23 minutes each. This represents more than ½ of our monthly clients in December.

The in-browser banner, a new media format to market, is a fixed position format that overlays the user’s browsing experience with an IAB standard format banner. This provides the opportunity for a brand to be a constant presence in a user’s browsing experience.
